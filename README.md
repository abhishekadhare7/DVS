# DVS
Data Visualization &amp; Storytelling (MBA-Business Analytics)
This Power BI dashboard provides a clear overview of retail and warehouse sales across major Indian cities from Jan 2023 to Oct 2025. It tracks key metrics like Total Sales, Profit, and Profit Margin, and highlights performance by Warehouse, Product Category, and Payment Mode.

Users can explore monthly sales trends, identify top-performing cities and products, and analyze customer segment preferences.
Built with Power Query and DAX, the dashboard transforms 150K+ transaction records into actionable business insights for smarter decision-making.
